INTELLIGENCE DATABASE

🔐 How do I verify my operator status?
Access the main terminal on the homepage, input command "1." and complete biometric authentication. Your tier level and clearance will be displayed instantly.
📱 How to install the PWA app?
On mobile: use the "INSTALL APP" button in footer. On desktop: look for the install icon in browser address bar. For manual: browser menu → "Add to Home Screen".
🎵 Audio not playing / flag not working?
Click the South African flag in bottom-left to activate background audio. National anthem plays on flag interaction. Ensure browser allows autoplay.
🏢 How to access branch documents?
Navigate to any branch card (Parables, HopCo, Stainless Steel, Lawk) and select "main entrance" for secure document portal. CIPC status shown.
⏰ Operator shift hours?
Each operator's shift is listed in their profile card on the main page. Contact COO for schedule modifications or special operations.
