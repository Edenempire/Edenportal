What is Node.js?

Node.js is an open-source, cross-platform JavaScript runtime built on Chrome's V8 engine. It uses an event-driven, non-blocking I/O model, perfect for scalable network applications.

  • Asynchronous & Non-blocking — handles many connections concurrently.
  • npm ecosystem — largest library of open-source packages.
  • CommonJS modules — require() and module.exports.
